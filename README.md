# ✅ TaskTracker <a name="about-project"></a>

> **TaskTracker** is the backend API, allowing users to manage their tasks.
> Besides of manaigng the simple name and task description, users can set the deadlines, performers of their tasks and change their tasks statuses.
> Users with the role "Group Moderator" can manage and change the details of all the tasks.
> Users with the role "Project Manager" can get the statistics of current amount of tasks and their statuses.
